Extract URLs from the text field and from the author website field on the WordPress comments.

== Description ==

WP Comments URLs Extractor is a easy to use WordPress plugin that allow users to extract URLs from the comments text and from the author website field. This is very useful if you want to extract all URLs from the comments in the Spam folder, and report these malicious URLs to the authorities or to other security services that keep tracks of malicious URLs.

Main features:

* Extract URLs from text field of Spam comments
* Extract URLs from text field of Trash comments
* Extract URLs from text field of Approved comments
* Extract URLs from text field of Unapproved comments
* Extract URLs from author website field of Spam comments
* Extract URLs from author website field of Trash comments
* Extract URLs from author website field of Approved comments
* Extract URLs from author website field of Unapproved comments

For more information visit the [plugin page](http://www.wpmize.com/wordpress-plugins/wordpress-plugin-wp-comments-url-extractor/).

== Installation ==

1. Place the 'wp-comments-urls-extractor' folder in your '/wp-content/plugins/' directory.
2. Activate WP Comments URLs Extractor.
3. Visit 'Comments URLs Extractor'.
